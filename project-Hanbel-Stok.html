Hanbel Stok

Kobiler İçin Stok Takip Otomasyonu

Proje Hakkında

Hanbel Stok Yönetimi; Kobilere Yönelik geliştirilmiş kapsamlı bir masaüstü ERP uygulamasıdır. Stok takibi, satış yönetimi ve cari hesap işlemlerini tek bir arayüzde birleştirerek işletmelerin günlük operasyonlarını hızlandırmayı hedefler.

Hızlı satış ekranı, barkod okuyucu desteği ve kategori bazlı ürün filtreleme gibi özellikler sayesinde kasa operasyonları minimum sürede tamamlanır. Lisans tabanlı aktivasyon sistemi ile yazılım güvenliği de ön planda tutulmuştur.

Temel Özellikler

  • Barkod okuyucu destekli hızlı satış ekranı
  • Kategori bazlı dinamik ürün filtreleme
  • Stok giriş / çıkış ve sayım yönetimi
  • Cari hesap ve borç / alacak takibi
  • Makine bazlı lisans aktivasyon sistemi
  • SQL Server tabanlı güvenli veri yönetimi

Teknolojiler

C# .NET Framework WinForms SQL Server Inno Setup
